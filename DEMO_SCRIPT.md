# TravelMind - Demo Script

## Recommended Demo Query
```
I want to go from New York to Tokyo in October for 7 days, budget $2000
```

## Recording a 90-Second Loom Video

### 0:00 - 0:10 | Intro
- Show the landing page
- Say: "This is TravelMind, an AI travel intelligence agent built with LangGraph and Gemini 2.0 Flash"

### 0:10 - 0:20 | Enter Query
- Type the demo query or click the preset button
- Click "ANALYZE"
- Say: "I type in my travel request in natural language and hit analyze"

### 0:20 - 0:50 | Agent Working
- Point to the Agent Activity panel on the left
- Say: "Watch the agent autonomously deciding which tools to call"
- Highlight each tool call as it appears:
  - "It's searching for flights..."
  - "Now checking hotels..."
  - "Getting weather data..."
  - "Checking visa requirements..."
  - "Finding activities and hidden gems..."
- Say: "This is real agentic AI - it's making decisions and calling tools on its own"

### 0:50 - 1:20 | Report
- Point to the report cards appearing on the right
- Say: "And here's the structured travel intelligence report"
- Scroll through the report highlighting:
  - Budget estimate
  - Flight prices
  - Hotel recommendations
  - Weather rating
  - Visa info
  - Top activities
  - The insider tip
- Say: "All of this data is from real-time web searches, not a static database"

### 1:20 - 1:30 | Close
- Say: "Built with Next.js, LangGraph, Gemini 2.0 Flash, and Tavily - all free tier. Deployed on Vercel."

## Tips for a Great Recording
- Use a dark browser theme to match the app aesthetic
- Zoom in to 125% so text is readable
- Start with a clean browser tab
- Let the agent run fully before scrolling the report
- Pause briefly on the insider tip - it's always interesting
